SuperPNG

SuperPNG is a Photoshop file format plugin with improved support for the PNG format. It is highly recommended for anyone creating textures or sprites with alpha channels (transparency) in Photoshop. It is a required tool for anyone creating images for TGE, TGEA, and TGB.

Older Is Better

The current version offered by Fnordware has, unfortunately, adopted Photoshop's style of using alpha layer transparency and ignoring the alpha channel. This is particularly problematic for TGEA materials, where the alpha channel may be used for special effects such as specular value. Fortunately, we have our own archive of the older version of SuperPNG available for download.
